# FOIL-Text-Header

Text Header is a DIV class that offers font sizes similar to Common HTML Tags like:

```html
<h1> <h2> <h3> <h4> <h5> <h6>
```

**To begin, call the div class named. FOIL-Text-Header.**

### Text Header 1

```html
<div class = "FOIL-Text-Header-1">This is the Text</div>
```

### Text Header 2
